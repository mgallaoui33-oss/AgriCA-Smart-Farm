# 🌿 AgriCA Smart Farm 🔴⚪
**Système de Gestion Agricole Intéligent avec IA**

Ce projet est une plateforme web interactive conçue pour moderniser la gestion d'une ferme de 2 hectares de fèves à **Majaz al Bab**.

### 🚀 Fonctionnalités principales :
- **Tableau de Bord :** Surveillance en temps réel de l'humidité du sol.
- **Contrôle de l'Irrigation :** Gestion à distance du sondage (Pompe) avec historique.
- **Scanner AI :** Diagnostic des maladies des plantes via l'IA (Gemini 1.5 Flash).
- **Interface Locale :** Support de la langue tunisienne pour une meilleure communication avec l'agriculteur.

### 🛠️ Technologies utilisées :
- **Backend :** PHP / MySQL
- **Intelligence Artificielle :** Python (API Gemini)
- **Frontend :** HTML5, CSS3 (Glassmorphism), JavaScript (AJAX)
- **Design :** Inspiré par les couleurs du **Club Africain** (Rouge et Blanc).
